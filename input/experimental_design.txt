Random forest:

necesito el código para crear en una libreta de jupyter un programa que utilice random forest, que cargue todos los módulos necesarios y que cargue también la información de 1000 cepas bacterianas para entrenar el modelo. Los datos que se van a cargar para entrenar el set son: mutaciones en 71 genes de resistencia, concentración mínima inhibitoria (MIC) de 5 antibióticos diferentes (IMI, AZT, FEP, MER y CIP) para esas cepas y la valoración clinica de esas MIC para cada cepa (es decir, si esa MIC para ese antibiótico se puede considerar resistente o sensible). Después de usar ese set de entrenamiento, se probará un set de test con 50 cepas y únicamente la información de las mutaciones en los 71 genes de resistencia y la salida esperada será una predicción de si cada mutación en el set de datos influye en la resistencia a cada uno de los antibióticos de las cepas disponibles. Además del codigo necesito una evaluación de si el modelo a utilizar (redes neuronales es el más adecuado para el problema y si no es así qué otros algoritmos seria recomendable usar). El dato de las mutaciones debe ser: el campo está en blanco si ese gen es igual que la cepa silvestre (wild type) o si no hay dato para ese gen, si la cepa tiene una o varias mutaciones en ese gen aparecen como una lista con los nombres de las mutaciones, separados por comas. Tambien necesito la estructura de los archivos de entrada de datos del programa, que significa cada columna y que formato debe tener, con un ejemplo de cada archivo. La entrada y la evaluación de los valores tiene que tener en cuenta que los valores de MIC pueden ser numéricos o llevar un signo (>, <, >=, <=) para refelejar que la concentración minima inhibitoria es mayor o menor de una concentración, pero no se conoce el valor exacto. Hay que tener en cuenta que en cada gen puede haber varias mutaciones y la presencia o ausencia de cada una de ellas, asi como la combinación de varias puede ser importante para la predicción, si el modelo no tiene en cuenta esto necesito que se ajuste para que lo tenga en cuenta. el archivo de entrada training_data.csv sea de la siguiente manera:
*primera columna: código identificador de la cepa
*2a a 7a: valores MIC de antibioticos y evaluaciones clinicas de resistencia o sensibilidad, que tienen que tener 3 categorías: sensible, resistente e indeterminado
*8a a final: información de mutaciones de cada gen, si no hay información sobre ese gen en la cepa el valor será un guion (-) y si no hay mutación not a number (nan)
